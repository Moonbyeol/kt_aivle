## 최빈값 확인

## 앞/뒤 값으로 채우기
![](https://i.imgur.com/OUwtjvA.png)
```python
# Ozone 변수 NaN 값을 바로 앞의 값으로 채우기
air['Ozone'].fillna(method='ffill', inplace=True)

# Solar.R 변수 NaN 값을 바로 뒤의 값으로 채우기
air['Solar.R'].fillna(method='bfill', inplace=True)

# 확인
air.isna().sum()
```

## 선형 보간법으로 채우기

