# 다중 회귀
- 여러독립변수가 종속변수에 영향을 미치는 선형 회귀
- y값을 설명하기 위해서는 여러 개의 x 값이 필요한 경우
	- 예: 여러 요인들에 의해 보스턴 지역 집 값이 결정 됨
- 회귀식:

```python
print(list(x_train))
print(model.coef_)
print(model.intercept)
```
